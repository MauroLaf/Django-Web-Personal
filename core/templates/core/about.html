Soy desarrollador backend con formación en administración de bases de datos y experiencia práctica en el desarrollo de aplicaciones utilizando Python y C#. He trabajado con bases de datos como MySQL, SQL Server y Oracle, aplicando criterios de diseño lógico, normalización y optimización de consultas. Además, utilizo herramientas como Swagger para documentar y probar APIs de manera profesional.
